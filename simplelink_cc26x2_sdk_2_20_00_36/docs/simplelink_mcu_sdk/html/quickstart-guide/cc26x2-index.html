Quick Start Guide for SimpleLink™ CC26x2 SDK¶

This guide serves as a road map for users developing applications and products using the TI SimpleLink CC26x2 wireless MCU platform. Whether a seasoned developer or just getting started, TI has created a variety of resources to simplify development on the CC26x2 platform. These resources will enhance your experience with the TI SimpleLink CC26x2 2.20.00.36 Software Development Kit (SDK) from the out-of-the-box demo to production.

This guide will walk you through the initial setup and demonstration of running your first sample application. The guide explains how to install the SimpleLink CC26x2 SDK and lists the various other tools required to get started with your first application. It will also suggest further resources to explore.
