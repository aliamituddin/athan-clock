Detailed Description

This module defines a toolchain abstraction layer through macros.

Usage:

typedef
OT_TOOL_PACKED_BEGIN
struct
{
char mField1;
union
{
char mField2;
long mField3;
} OT_TOOL_PACKED_FIELD;
} OT_TOOL_PACKED_END packed_struct_t;
