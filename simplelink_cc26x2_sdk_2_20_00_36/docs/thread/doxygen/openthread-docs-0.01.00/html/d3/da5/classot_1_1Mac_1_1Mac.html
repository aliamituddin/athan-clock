uint16_t  aScanDuration,
ActiveScanHandler  aHandler,
void *  aContext 
)

This method starts an IEEE 802.15.4 Active Scan.

Parameters
[in]aScanChannelsA bit vector indicating which channels to scan. Zero is mapped to all channels.
[in]aScanDurationThe time in milliseconds to spend scanning each channel. Zero duration maps to default value kScanDurationDefault = 300 ms.
[in]aHandlerA pointer to a function that is called on receiving an IEEE 802.15.4 Beacon.
[in]aContextA pointer to arbitrary context information.
Return values
OT_ERROR_NONESuccessfully scheduled the Active Scan request.
OT_ERROR_BUSYCould not schedule the scan (a scan is ongoing or scheduled).

This method starts an IEEE 802.15.4 Energy Scan.

Parameters
[in]aScanChannelsA bit vector indicating on which channels to scan. Zero is mapped to all channels.
[in]aScanDurationThe time in milliseconds to spend scanning each channel. If the duration is set to zero, a single RSSI sample will be taken per channel.
[in]aHandlerA pointer to a function called to pass on scan result or indicate scan completion.
[in]aContextA pointer to arbitrary context information.
Return values
OT_ERROR_NONEAccepted the Energy Scan request.
OT_ERROR_BUSYCould not start the energy scan.

§ IsInTransmitState()

bool ot::Mac::Mac::IsInTransmitState ( void  )

This method returns if the MAC layer is in transmit state.

The MAC layer is in transmit state during CSMA/CA, CCA, transmission of Data, Beacon or Data Request frames and receiving of ACK frames. The MAC layer is not in transmit state during transmission of ACK frames or Beacon Requests.
