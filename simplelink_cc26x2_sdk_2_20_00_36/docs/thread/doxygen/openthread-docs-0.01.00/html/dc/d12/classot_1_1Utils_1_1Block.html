Detailed Description

This class represents a memory block.

A block is of the structure as below. 

+------------------------------+
| mSize   |  mMemory |  mNext  |
|---------|----------| --------|
| 2 bytes |  n bytes | 2 bytes |
+------------------------------+

Since block metadata is of 4-byte size, mSize and mNext are separated at the beginning and end of the block to make sure the mMemory is aligned with long.
