Overview¶

The TI-OpenThread Stack is part of the SimpleLink CC26x2 SDK, which contains the supported software for the CC26x2. The TI-OpenThread Stack is based on the OpenThread stack by Nest with the added benefit of running on top of TI-RTOS. Please refer to the Software Architecture section for more information about the TI-OpenThread Stack implementation.

Note

Do not use this document as a substitute for the complete protocol specification. For more details, see the OpenThread Wiki and the Thread specification

Introduction¶

The SimpleLink CC26x2 SDK is part of TI’s SimpleLink MCU platform which offers a single development environment that delivers flexible hardware, software and tool options for customers developing wired and wireless applications. For more information about the SimpleLink MCU platform, visit (http://www.ti.com/simplelink).

Thread is an IP-Based mesh networking protocol that runs on IEEE 802.15.4 radios. Thread was designed with one goal in mind: “To create the best way to connect and control products in the home”.

OpenThread is an open source implementation of the Thread specification, which includes all Thread networking layers and device roles, as well as Border Router support.

Besides supporting all the OpenThread features the TI-OpenThread Stack has the added benefit of running on top of TI-RTOS which inherently adds support for all the LaunchPad ecosystem and peripheral drivers.
